Network security has become an increasingly critical concern in modern computing, yet the tools available for understanding network traffic remain largely inaccessible to those without expert-level knowledge. This project addresses that gap by developing the Network Traffic Visualiser, a Python-based application designed to capture, analyse, and display live network packets through real-time visual graphs and structured terminal output. The system captures Ethernet frames and dissects IPv4, TCP, UDP, and ICMP protocols using Python's socket and struct libraries, presenting traffic data in a clear, interpretable format without requiring prior expertise in packet analysis.
The project was managed using a Kanban methodology, with development carried out in a Kali Linux environment using Visual Studio Code. The application was evaluated against Wireshark, an industry-standard packet analyser, demonstrating comparable data extraction across all supported protocols. Colour-coded visualisations, configurable capture filters, and real-time chart updates were implemented in accordance with established information visualisation principles, including Shneiderman's overview-first design philosophy.
The tool successfully meets its core objectives of accessibility, clarity, and real-time network monitoring, offering a foundation for future development into a fully featured, GUI-based educational network analysis platform.
